STL Model Viewer

STL File Preview with JSC3D Tool Kit

This is an implementation of JSC3D, a canvas-based Javascript 3D Library.
Current functionality allows file exchange via drag-and-drop into 3D Viewing Window.

    User Interface
  • Zoom In/Out:
    1. Scrool Wheel
    2. Shift + LMC
  • Pan:Ctrl + LMC
  • 3D Rotation: LMC (Hold+Drag)
    • LMC = Left Mouse Click
