STL Model Viewer

STL file preview with JSC3D

This is an implementation of JSC3D, a canvas-based Javascript 3d library.

The original functionality includes a drag-and-drop feature for the 3d model viewer, allowing a user to bring their own .STL files into the viewer.

This drag-and-drop feature can been snipped, as it may not provide the end user with any necessary functionality.

JSC3D allows models to be rotated and zoomed in/out: drag with your left mouse button in the canvas for rotation, mouse-wheel to zoom.

Try to drag-and-drop ortho bracket STL files to see how they come out.
